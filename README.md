# Game_2 – my first JavaFX game

A simple 2D side‑scroller I wrote during my **2nd year** of university.  
This project was my personal way to get familiar with **JavaFX**, animations, and keyboard input handling.

> I’m now in my 3rd year – this code reflects my beginner skills from ~1 year ago.  
> It’s not perfect, but it’s an honest start.

## What it does

- Infinite horizontally scrolling background  
- Move left/right (`A` / `D`)  
- Jump (`SPACE`)  
- Pause (`ESCAPE`)  
- A moving enemy – collision ends the game  

## Technologies

- Java 23 (works with 17+ too)  
- JavaFX 17  
- Maven (with Maven Wrapper)

## Run it

```bash
mvnw.cmd clean javafx:run
```
